# 🌱 Smart Irrigation System with AI (ESP32 + IoT)

## 🚀 Overview
This project is an intelligent irrigation system that automates plant watering based on soil moisture and daylight conditions. It is enhanced with AI integration (Gemini API) and IoT control using Blynk.

## 🔥 Features
- Soil moisture-based irrigation
- Day/Night detection using LDR
- Automatic + Manual control
- Blynk IoT remote monitoring
- AI integration for smart responses and control
- Water-saving optimization

## 🧠 Working
The system checks:
- Soil moisture level (< 40%)
- Daylight condition (via LDR)

Only if both conditions are satisfied:
➡️ Water pump turns ON

AI integration allows smart interaction and decision support.

## 🛠️ Components Used
- ESP32
- Capacitive Soil Moisture Sensor
- LDR
- MOSFET (IRLB4132)
- 12V Water Pump
- Blynk IoT Platform

## 💡 Applications
- Smart agriculture
- Home gardening automation
- Water conservation systems
